# Kashyap's Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and experience. Built with **Next.js 13** and **Tailwind CSS** for a fast, accessible, and visually stunning web experience.

## 🌟 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Dark/Light Mode**: Seamless theme switching with `next-themes`
- **Smooth Animations**: Elegant scroll animations for an engaging user experience
- **Performance Optimized**: Built with Next.js 13 for optimal speed and SEO
- **TypeScript**: Type-safe development for better code quality
- **Modern Styling**: Tailwind CSS for rapid and customizable UI development

## 📁 Project Structure

```
├── app/                      # Next.js app directory

```

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ and npm installed

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd tailwindcss-and-nextjs-portfolio-final
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view your portfolio in the browser.

## 📦 Available Scripts

- `npm run dev` - Start the development server (hot reload enabled)
- `npm run build` - Build the application for production
- `npm start` - Start the production server
- `npm run lint` - Run ESLint to check code quality

## 🎨 Customization

To make this portfolio truly yours, customize the following:

- **HeroSection**: Update with your name, title, and call-to-action
- **AboutSection**: Add your bio, skills, and professional background
- **ProjectsSection**: Replace with your featured projects and descriptions
- **Footer**: Update contact information and social media links
- **Colors & Fonts**: Modify `tailwind.config.js` for your brand colors and typography
